## :question: What is this Repository about?

This project contains the example code shown for performing API Testing with [Playwright TypeScript](https://playwright.dev/docs/api-testing)

## :briefcase: What does this repo contain?
- Example API Tests with Playwright TypeScript
- [restful-ecommerce APIs](https://github.com/mfaisalkhatri/restful-ecommerce) is used as a demo application to write API tests.

## :hammer_and_wrench: Talking more about the Scenarios Covered in this project:
You will get the answers to the following questions and its respective working code example with [Playwright Java](https://playwright.dev/java/docs/api-testing#writing-api-test) framework in this repository:
- How to write tests for `GET` requests?
- How to verify the Response Status Code?
- How to verify the Response Body?
- How to attach Response data to the Playwright Report?
- How to test POST API requests?
- Builder Pattern + Faker in API Test Automation using Playwright with TypeScript
